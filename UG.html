Назначение программы

Программа "Управление базой данных" предназначена для эффективного управления информацией о сотрудниках, отделах, вакансиях и заявках на вакансии. Разработанная с использованием современных технологий информационных технологий, данная программа обеспечивает удобное добавление, редактирование и удаление записей, а также предоставляет функционал поиска для упрощения работы с базой данных.

Системные требования

Прежде чем начать использовать программу, убедитесь, что ваш компьютер соответствует следующим системным требованиям:

  • Операционная система: Windows 7/8/10/11, macOS, Linux
  • Python 3.6 и выше установленный на вашем компьютере
  • Библиотеки Tkinter, SQLite3, Pillow (для отображения изображений)

Выполнение программы

  1. Запуск программы:
    • Запустите Python-скрипт app.py, чтобы открыть главное окно приложения.
  2. Выбор таблицы:
    • Нажмите на соответствующую кнопку ("Сотрудники", "Отделы", "Вакансии", "Заявки на вакансии") для открытия окна с данными выбранной таблицы.
  3. Работа с записями:
    • В каждом окне вы можете использовать кнопки "Добавить", "Редактировать" и "Удалить" для управления записями.
    • Для редактирования выберите запись и нажмите "Редактировать". Затем внесите необходимые изменения и сохраните.
  4. Поиск данных:
    • Используйте поле ввода для поиска, чтобы быстро найти нужные записи.
  5. Экспорт данных:
    • Вы можете экспортировать данные в Excel, используя кнопку "Экспорт в Excel".
  6. Справочники:
    • В главном окне также предоставлены кнопки для просмотра данных справочников ("Пол", "Образование", "Должности", "Профессии").

Пример использования программы

Добавьте нового сотрудника, указав его ФИО, пол, дату рождения, образование, должность, профессию, дату поступления на работу, оклад, паспортные данные и адрес.

Откройте окно "Отделы" и создайте новый отдел, присвоив ему название и указав руководителя.

Перейдите к окну "Вакансии", чтобы добавить новую вакансию с требованиями и условиями работы.

В разделе "Заявки на вакансии" оставьте заявку на интересующую вакансию.

Используйте функцию поиска для быстрого доступа к нужным записям.

Версия программы в формате .exe

Если вы используете версию программы в формате .exe, убедитесь, что все необходимые файлы находятся в одной директории:

  • app.exe - исполняемый файл программы
  • employee_db.db - файл базы данных
  • icon.png - иконка программы

Примечание: Перед использованием убедитесь, что все необходимые библиотеки Python установлены на вашем компьютере. Следуйте инструкциям в программе для сохранения изменений и закрытия окон.
